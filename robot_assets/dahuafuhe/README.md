# Dahuafuhe Asset Bundle

This directory contains the standalone `dahuafuhe` robot asset bundle.

Contents:

- `start.launch.yaml`: launch config for the dahuafuhe bundle
- `robot/rokae_cr7_dahuafuhe.yml`: CuRobo2 robot config
- `robot/curobo/rokae_cr7_dahuafuhe.urdf`: URDF
- `robot/curobo/meshes/`: robot and tool meshes
- `robot/spheres/rokae_cr7_dahuafuhe_spherized.yml`: collision-sphere config
- `bundle_manifest.json`: materialization summary

Notes:

- this directory only describes the dahuafuhe asset bundle itself
